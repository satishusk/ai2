Система нечеткого вывода.

Входные переменные:
  Цена транспортного средства {
    Дешево[20, 400),
    Средне[300, 6_000),
    Дорого[5_000, 100_000]
  },
  Качество транспортного средства {
    Плохое[0, 4),
    Среднее[3, 7),
    Высокое[6, 10]
  }

Выходные переменные:
  Спрос на транспортное средство {
    Низкий[0, 4),
    Средний[3, 7),
    Высокий[6, 10]
  }

Правила:
 Если:
 (цена низкая && качество высокое)
  ||
 (цена средняя && качество высокое)
  ||
 (цена низкая && качество среднее),
 то [спрос высокий];

 Если:
 (цена средняя && качество среднее)
  ||
 (цена низкая && качество низкое)
  ||
 (цена высокая && качество высокое)
 то [спрос средний];

 Если:
 (цена высокая && качество среднее)
  ||
 (цена высокая && качество низкое)
  ||
 (цена средняя && качество низкое)
 то [спрос низкий];
